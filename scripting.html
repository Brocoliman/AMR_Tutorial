Intro to Scripting

This tutorial assumes that you already know how to make a basic Marble Race. If not, go the tutorials page . When the code given has an asterisk(*) in a part, do not type the asterisk; instead follow the direction in the asterisk after the code. Note that the tutorials below are just a foundation. The contraptions you make are actually based off all of them. This is the Intermediate level, Scripting.
To go in an object's script, go along with the following steps:
  1. Go in the Menu
  2. Find the Script Menu
  3. Don't get nervous from the size of the script; You'll soon get familiar with it

Basic Functions

Notes:

  • Some of the sections are conditions, for example, onCollide, that triggers a certain code, while some of them are variables of the object, for example, the text
  • The onCollide code is triggered when another object touches it
  • The update code is triggered every 60th of a second
  • You can create variables in the top bar below the scripting description
  • A variable lock can be used so an object, especially teleporters that teleport other teleporters, does not get any of its attributes changed. They can be made by putting a certain variable set in the update section
  • For more tips regarding the script, check out the Algodoo Scripting Wiki

Teleporter

Teleporters are the most important things in Marble Races. You may not even see it in action; it might be in the backstage!
To make a simple, go along with the following steps:
  1. Make 2 custom sized shapes
  2. Go in ones script menu, and in onCollide, in the "(e)=>{}"(you never put your code outside of that, unless the section doesn't have it), type: e.other.pos = my.*var1
  3. Go in the other object's script, and in the postStep, write: my.*var1 = pos
  4. *The code after the asterisk is a variable, so you can customize the name, as long as both are same
  5. The teleporter will only work if an object goes in the right one, which is the one coded in Step 2. Also, the teleporter won't work on an object that doesn't have any common collision layer as the teleporter. It is best to cancel out all the collision layers of the Step 3 rectangle
  6. There!! Try it out ;)

Velocity Pads

Same as teleporters, lots of backstage action. These are unarguably the most used element in Marble Racing.
To make a Simple Track, go along with the following steps:
  1. It is preferred to use rectangles than the brush, so make a long rectangle and color it with your marble race theme. If there is no theme, use black(for everything besides the marbles and transparent stuff).
  2. Try making all the track rectangles the same height
  3. *For Later reference*If you change the collision layer of the marbles, make sure the track also has that layer checked, unless you don't want the marble to collide with it
To make a Bowl, go along with the following steps:
  1. Make a rectangle(size is on you)
  2. Get the Eraser Tool, and set the size of it to how big you want the inner bowl to be
  3. From top to near bottom of the rectangle, hold shift while dragging down with the eraser tool, and make sure it is completely vertical
  4. Set the size of the eraser to slightly bigger(not too much) than the marbles
  5. Use this eraser to carve a hole at the bottom of the bowl

Spinners

Spinners are used throughout every marble race. From Classic Marbles Races to fully themed Marble Racees, spinners is an important part of Marble Races to be able to make.
To start, go along with the following steps:
  1. Make a custom sized rectangle(The height should be 5 times the width)
  2. Hold Ctrl or Command, based on your computer, and then click on the rectangle. This creates a copy of it
  3. Click(yep, only click, do not select)the rectangle, then Turn it 90 degrees. The original rectangle below should stay where it was at the beginning
  4. Now you should have a plus sign. These next steps are very important.
  5. From the plus sign, select the rectangle that has overlaps the other, and go in the menu. Then, click on Combine Shapes, and after that, click Add. Then exit the menu. Now, delete the rectangle that overlaps the other. You will now see just a plus sign, no overlaps
  6. Enter the plus sign menu, and go to Geometry Action. Click on add axle. The Axle will be shown on the plus sign(now about to be a spinner)
  7. Go in the Axle menu, and turn the "a"(In the HSVA slider section) in the Appearance section to 0
  8. Go to the Axle section, and click the check on Motor. 2 sliders will appear, and set the one that says Motor Torque to the max(or maybe 100 ^ 1000, which sets it to infinite). The Motor Speed is to your choice

Trampolines

Trampolines, otherwise known as bounce pads, are not used very much. Instead, velocity pads are used more, and are more advanced. But, hey, it is still good to learn something than to not.
To start, go along with the following steps:
  1. Make a custom sized rectangle
  2. Go in the Menu, then to Material. Set the restitution to a custom value. 1000 makes an object bounce on it, with the same bounce height every time

Text

Words are used commonly throughout Marble Races. That's why you should know how to make them!
To start, go along with the following steps:
  1. Make a custom sized rectangle
  2. Go in the Menu, and in the Text section, add the words you want in the big box
  3. You can change the color of the text at the color selection below the box
  4. Maybe your text looks a bit too small, so change the font size to your desire
  5. Maybe your texts looks a bit too pixelated, so change the font resolution to the max

Conveyor

Conveyors are commonly used in basic marble races. That's all I can say:)
To start, go along with the following steps:
  1. Make a custom shaped spinner(Oval and square ones are recommended)
  2. Make a copy of the spinner, and put it next to the original
  3. Set the collision layer of the 2nd different from the first. This prevents the 2 spinners from getting stuck on each other. If your 2 spinners are too far away to collidee with each other, move them closer.
  4. Turn the 2nd spinner so it is pointing 90 degrees right of the 1st.
  5. Make a copy of both(I recommend to copy them at the same time to make it neat), and put the copy next to the original 2. Do the distance check shown in step 3. Now you should have 4 spinners
  6. Make as much copies as you want, and remember to distance check them.

Controller(*Important!*)

Controllers are used to start a section of a marble race, by hand. After making one, press the trigger key and the block will disappear, lettin any Marble above it start the next section of your race.
To start, go along with the following steps:
  1. Make a custom shape to be destroyed
  2. Go in the objects Menu, and go in Controller. Set the destroy key by clicking on "[none selected]"

Piston **!!!More Important!!!**

Pistons may not sound or look important at first, but it is used to move an object back and forth, while using a script to make an object do the same thing is just a little too complicated to explain. Note: The steps of how to make it written by me down below is using my best explaining abilities.
To start, go along with the following steps:
  1. Make a custom sized rectangle, with the height 5 times as much as the width. Make sure to glue it to the background
  2. Make a copy of it and put the copy next to the original, horizontally. Space the distance between them as the width of the moving object
  3. Make a custom object(this will be the moving object), and fit it between the rectangle created on the previous step. Do not glue it to the background
  4. Make a custom sized circle. The circumference of the circle will be how far the moving object moves. Do not glue it to the background
  5. Use a brush, or make a rectangle, that overlaps the moving object on one side, and the circle on the other. Do not glue it to the background
  6. Put an axle with the axle tool on the space where the bar overlaps the circle, and where it overlaps the moving object. If you make a motor on it, good luck with seeing your piston wobble back and forth right when you run the simulation
  7. Go in the menu for the circle, and add a center axle in Geometry Actions. Set the Motor Torque very high, and the speed is for you to decide.
  8. *One last step* Make sure that the moving object and the rectangles made in step 1 have at least 1 common collision layer
  9. ...aaannd your done!!! It should work like a piston now
